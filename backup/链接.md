## 项目说明
全部站点基于 Gmeek 模板实现：
- 写作方式：GitHub Issues 撰写 Markdown，打上 `document` 标签自动发布
- 部署方案：GitHub Actions + GitHub Pages，完全免费、All in GitHub
- 设备适配：手机浏览器即可完成全部写作、更新操作
- 支持特性：代码块、Mermaid 流程图、图片粘贴、表格、评论、深色模式

## 通用使用流程
1. Fork Gmeek 模板仓库
2. 仓库名称改为 `用户名.github.io`（根站）或自定义子目录
3. 开启 GitHub Pages，部署源选择 GitHub Actions
4. 新建 Issue，书写 Markdown 内容，添加 `document` 标签
5. 提交后自动构建，几分钟后访问对应域名查看文章
